Project Overview

AluminiNext is a full-stack community platform designed to supercharge alumni networking. The application creates a vibrant ecosystem for graduates with features like real-time chat for instant communication, lively group collaboration spaces, and seamless event planning tools. It also provides direct career support through a curated job board and offers administrators at-a-glance oversight with real-time insight dashboards.

The platform is built on a robust backend using Node.js and Express.js, with MongoDB managing the data. User authentication and security are handled by Passport.js, ensuring that all interactions are secure and private. The real-time chat and notification features are powered by WebSockets, providing an interactive and engaging user experience, while the front end is rendered dynamically using EJS and styled with Bootstrap for a clean, responsive interface.
