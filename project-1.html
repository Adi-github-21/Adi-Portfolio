Wanderlust

This full-stack Airbnb clone provides a seamless platform for users to discover, book,and review unique properties around the world. Built with Node.js and Express, it features a robust RESTful API for dynamic property management, secure bookings, and cloud-based image hosting.

Project Overview

Wanderlust is a dynamic, full-stack web application designed to replicate the core experience of Airbnb. It empowers users to seamlessly list their own properties, explore a wide range of unique stays, and make secure bookings. The platform fosters a trustworthy community through an integrated user review system, allowing both hosts and guests to share their experiences and build credibility.

The application is built on a robust backend using Node.js and Express.js, featuring a comprehensive set of RESTful APIs for all CRUD operations. It includes role-based access control to differentiate between users and property owners, with secure image hosting managed through the Cloudinary API. This architecture ensures a scalable, secure, and feature-rich environment for a modern property rental marketplace.
